# HFT-with-FPGA

This repository contains code from HFT with FPGA project. This includes:
### Vitis:
- echo.c (modified from lwIP echo server)

### Vivado:
- hft.v (combined IP for MACD and RSI calculation)

### Python Scripts:
- client.py (to send input data over ethernet to FPGA)
- data.csv (historical data from Nifty 50)
